## 🔧 Teknik Detaylar

### Auto-refresh (15s)
```tsx
useEffect(() => {
  if (!wsConnected) {
    const interval = setInterval(() => {
      if ((predQ as any)?.refetch) {
        (predQ as any).refetch();
        setLastUpdated(new Date());
      }
    }, 15000); // 15 seconds
    return () => clearInterval(interval);
  }
}, [wsConnected, predQ]);
```
